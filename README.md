# Farias Estofados

Projeto frontend em React + Vite para a loja Farias Estofados.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Observacoes

- Projeto preparado para desenvolvimento local independente.
- Ajuste os componentes, paginas e integracoes conforme a evolucao do app.
